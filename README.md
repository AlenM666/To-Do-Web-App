# To-Do-Web-App 📝
<br>

### This is a super simple Todo App built using JS. 


<br>

Features: 
- [x] Add Task
- [x] Delete Task
- [ ] Save todos to Local Storage when the state changes
- [ ] Load the todos from Local Storage when the site is loaded again
